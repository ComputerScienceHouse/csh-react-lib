# csh-react-lib
CSH Themed React Library to make frontend applications easier to stand-up.

## Installation
```sh
npm install csh-react-lib
```

## Usage
```js
import { Button } from 'csh-react-lib';
...

<Button kind="primary" onClick={() => console.log('Hello World'))} />
```
